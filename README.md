# Claude Code AI研究工作流入門指南
## 2026年6月10日 國立清華大學環境與文化資源學系

### 課程介紹
本講義為清大環文班夜間線上課程教材，系統性介紹如何使用Claude Code進行學術研究的全流程。涵蓋從文獻探索、議題分析到研究寫作的完整AI研究工作流。

### 核心主題

#### 📚 **文獻探索**
- 善用AI進行文獻搜尋與篩選
- 快速理解學術論文核心內容
- 建構研究主題的文獻基礎

#### 🔍 **議題分析**
- 識別研究議題與研究缺口
- 進行多角度議題分析
- 提出有意義的研究問題

#### ✍️ **研究寫作**
- 結構化組織研究內容
- 提升寫作效率與品質
- 從初稿到成稿的優化流程

### 課程特色
- **實踐導向**：兼具理論與實操示範
- **工具應用**：深入介紹Claude Code的研究應用
- **流程整合**：展示研究工作流的完整銜接
- **環境議題**：聚焦環境與社會議題研究

### 適用對象
- 環境研究相關領域學生
- 準備從事學術研究的研究者
- 想掌握AI助研技能的人士

### 使用方式
1. 按順序閱讀各個課程資料
2. 配合實例理解Claude Code應用
3. 自行測試工作流程
4. 修改參數進行客製化研究

### 課程資源
- `assets/` - 課程所需的輔助資料與圖表
- `index.html` - 課程線上版本
- `content.enc` - 加密內容檔案
